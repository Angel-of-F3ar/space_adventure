# Space Adventure

A text-based Python game where you wake up on a dark, damaged spaceship and make choices to survive. Your decisions affect the outcome â€” will you escape, or perish among the stars?

## Features

- Branching storylines based on player input
- Simple command-line interface
- Replayable with multiple endings
- Written in clean Python

## How to Play

1. Clone the repo:
https://github.com/Angel-of-F3ar/space_adventure.git

2. Navigate into the project folder:
cd space_adventure

3. Run the game:
python space_adventure.py
